# IAOS(Intelligent analysis of securities)证券智能分析系统

---
#### IAOS是一款Python开发的中低频率智能证券量化分析系统
- 主要指导思想：价值投资为主，技术分析为辅
- 侧重点为基本面选股、市场行为选股，趋势择时、牛熊线择时

---
#### IAOS提供以下主要功能
- 量化选股
- 量化择时
- 条件选股
- 个股量化分析
- 行业分类、周期非周期股分类、大、中小盘股分类
- 侦测大盘周期、情绪
